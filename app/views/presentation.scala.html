Recipe Manager with Play Framework

This site serves to manage recipes and ingredients through requests.

The management options are as follows:

-You can create an ingredient and enter it in the database

-You can get an ingredient by name

-You can get the list of ingredients by category and by shop name

You can get the list of all ingredients

-You can update an ingredient or remove it

-You can create a recipe

-Introduce an ingredient in a recipe

You can get the list of all recipes

-You can get a recipe by name and get the list of vegetarian recipes

-You can update or delete a recipe

-Finally, you can get a shop by name
